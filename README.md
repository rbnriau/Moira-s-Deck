# Moira's Deck
- **Genero:** Juego narrativo generativo local con mecánica de cartas.
- **Metafora central:** Las Nornas/Moiras (tejedoras del destino) te ofrecen al posibilidad de cambiar tu destino jugandotelo a las cartas.
- **Intro cinematografica:** Texto definitivo pulido: "Tu destino está tejido y es... terriblemente aburrido. He barajado estas cartas. Tómalas si te atreves. Tira de un hilo. Elige una carta. Veamos si logras tejer una leyenda o si te enredas en tu propia ruina."
- **Estética Visual:** Dark Fantasy nórdico
